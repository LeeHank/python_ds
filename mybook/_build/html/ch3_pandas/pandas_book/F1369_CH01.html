把 運算符 改成 method, 可用方法¶

  • 數值運算：

    • + -> .add.

    • - -> .sub

    • * -> .mul.

    • / -> .div.

    • // -> .floordiv.

    • % -> .mod.

    • ** -> .pow

  • 比較運算:

    • < -> .lt.

    • <= -> .le.

    • > -> .gt.

    • >= -> .ge.

    • == -> .eq

    • != -> .ne
